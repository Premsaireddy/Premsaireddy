### 📉 Customer Churn Analytics Dashboard
> Python · scikit-learn · XGBoost · SMOTE · Streamlit · Plotly

End-to-end churn prediction system for a telecom dataset — from raw data to a live, deployed web dashboard. Trained and compared three ML models, handled class imbalance with SMOTE, and built an interactive Streamlit app for live predictions and customer risk exploration.

- Trained Logistic Regression, Random Forest, and XGBoost models; selected best model based on ROC-AUC performance
- Applied SMOTE to handle class imbalance in the churn target variable
- Built interactive dashboard with ROC curves, confusion matrices, feature importance, and live churn prediction input
- Segmented customers into risk tiers (Low / Medium / High) for actionable business insights
- Deployed live on Streamlit Cloud — accessible without any local setup
